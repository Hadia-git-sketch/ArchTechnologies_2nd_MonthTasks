# 📊 Arch Technologies Internship - Month 2
## Machine Learning & Predictive Modeling Solutions

This repository contains the core projects developed during the second month of my internship at **Arch Technologies**. These tasks focus on implementing the end-to-end Machine Learning lifecycle, from Exploratory Data Analysis (EDA) to interactive model deployment.

---

## 🏗️ Project 1: California Housing Price Predictor (Task 3)
### **Objective**
To build a regression model that predicts the median house value in California blocks based on 1990 Census data, including geographical and socioeconomic factors.

### **Technical Highlights**
* **Algorithm:** Linear Regression
* **Key Features:** Median Income, House Age, Average Rooms, and Population.
* **Preprocessing:** Implemented `StandardScaler` to normalize feature scales for better model convergence.
* **Visualization:** Generated Correlation Heatmaps and Geographical Scatter Plots to identify high-value coastal clusters.
* **Interface:** A professional Gradio UI for real-time price estimation.

## 🌸 Project 2: Iris Flower Species Classifier (Task 4)
### **Objective**
To develop a multi-class classification system that identifies Iris flower species (*Setosa, Versicolor, Virginica*) based on sepal and petal morphology.

### **Technical Highlights**
* **Algorithm:** Random Forest Classifier (Ensemble Learning)
* **Features:** Sepal Length/Width and Petal Length/Width.
* **Insights:** Utilized Feature Importance analysis to determine that **Petal Width** is the most significant biological indicator for species identification.
* **Performance:** Achieved high precision and recall, validated through Confusion Matrices and Classification Reports.
* **Interface:** A themed Gradio dashboard providing class predictions with real-time **Confidence Scores**.
